Classroom Automation and Weather Monitoring System Using ESP32

Overview

This project is an IoT-based classroom automation system developed using ESP32. The system automatically controls lights and fans based on motion, temperature and light intensity while monitoring environmental conditions in real time.

---

Features

- Automatic Light Control using LDR
- Motion Detection using PIR Sensor
- Temperature Monitoring using DHT11
- Humidity Monitoring using DHT11
- OLED Real-Time Display
- Relay Controlled Fan and Light
- Adjustable Temperature Threshold
- Energy Efficient Operation

---

Components Used

- ESP32 Development Board
- DHT11 Temperature and Humidity Sensor
- PIR Motion Sensor
- LDR Sensor Module
- SSD1306 OLED Display
- 2 Channel Relay Module
- Push Buttons
- Jumper Wires
- Power Supply

---

Working

1. PIR sensor detects motion inside the classroom.
2. LDR sensor detects ambient light conditions.
3. DHT11 measures temperature and humidity.
4. ESP32 processes sensor data.
5. Fan is controlled according to temperature threshold.
6. Lights are controlled according to motion and light conditions.
7. OLED continuously displays system status.
